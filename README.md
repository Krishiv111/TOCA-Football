# TOCA Football Player Portal

A full-stack web application that allows players to view their past training sessions, upcoming appointments, and profile stats.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React + TypeScript + Vite |
| Styling | TailwindCSS v4 + Shadcn |
| Routing | React Router v6 |
| Backend | Node.js + Fastify + TypeScript |
| Data | JSON files (no database) |
| Dev tooling | tsx watch, concurrently |

---
